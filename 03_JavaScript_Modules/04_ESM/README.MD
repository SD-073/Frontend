|                   | Default Export      | Named Export        |
| ----------------- | ------------------- | ------------------- |
| How many per file | Only **one**        | As many as you want |
| Import name       | Any name you choose | Must match exactly  |
| Curly braces      | ❌ No               | ✅ Yes              |

> **Prefer named exports.** With named exports, you must use the exact name, which makes it harder to make mistakes and easier to search for things in your codebase.
